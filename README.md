# BSU-test-master

Приложение на Python с графическим интерфейсом (Tkinter) для извлечения тестов из Word-документов и последующего прохождения тестов.

## Возможности
- Извлечение тестов из таблиц Word (.doc/.docx).
- Поддержка спецсимвола для определения правильного ответа.
- Сохранение извлечённых тестов в JSON с именем исходного файла.
- Настройки тестирования (случайный порядок, количество вопросов, только нерешённые).

## Установка
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск
```bash
python app/main.py
```

Для работы с `.doc` требуется установленный LibreOffice (`soffice`).
